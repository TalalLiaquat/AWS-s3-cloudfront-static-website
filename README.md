# 🚀 AWS Static Website Hosting using S3 & CloudFront

## 📌 Project Overview
This project demonstrates the deployment of a fully functional static website using **Amazon Web Services (AWS)**. The website is hosted on **Amazon S3** and delivered globally using **Amazon CloudFront CDN**, ensuring high availability, scalability, and improved performance.

The main goal of this project was to understand real-world cloud deployment, content delivery optimization, and basic DevOps practices using AWS services.

🔗 CloudFront URL: https://d28q1emyk5ygal.cloudfront.net/

## 🏗️ Architecture
The project follows a simple cloud architecture:

User → CloudFront (CDN) → S3 Bucket → Static Website Files (HTML/CSS)
